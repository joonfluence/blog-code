### 자주 쓰는 단축키 활용

1. 반복되는 코드 한번에 수정하기 : CMD(CTRL) + D
2. 변수명/함수명 한번에 바꾸기
   - Mac : Ctrl + F2
   - Windows : F2
3. 코드 위/아래로 복사하기 : opt + Shift + ↑,↓
4. 여러 곳에 커서를 두고 수정하기 : Alt + 마우스클릭
5. 코드 맨 마지막에 커서 두기 : Shift + Alt + I
6. 사이드바 숨김/보이기 : CMD(Ctrl) + B
7. 이전커서로 이동 : CMD(CTRL) + U
8. 함수/변수 따로 빼기 : CMD(Ctrl) + Shift + R
