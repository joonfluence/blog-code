|:—:|:—:|

|Cell1|Cell2|

헤더 셀을 구분할 때 3개 이상의 **`-`**(hyphen/dash) 기호가 필요합니다. 

헤더 셀을 구분하면서 **`:`**(Colons) 기호로 셀(열/칸) 안에 내용을 정렬할 수 있습니다. 

가장 좌측과 가장 우측에 있는 **`|`**(vertical bar) 기호는 생략 가능합니다.
