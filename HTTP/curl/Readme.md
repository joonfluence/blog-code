### Curl 사용하는 방법

- 정의 : 다양한 통신 프로토콜을 이용하여 데이터를 전송하기 위한 라이브러리와 명령 줄 도구를 제공하는 컴퓨터 소프트웨어 프로젝트
- 활용 : SHELL에서 REST API 테스트를 하고 싶을 때 사용
- 지원 프로토콜
  - DICT
  - FILE
  - FTP
  - FTPS
  - GOPHER
  - GOPHERS
  - HTTP
  - HTTPS
  - IMAP
  - IMAPS
  - LDAP
  - LDAPS
  - MQTT
  - POP3
  - POP3S
  - RTMP
  - RTMPS$$
  - RTSP
  - SCP
  - SFTP
  - SMB
  - SMBS
  - SMTP
  - SMTPS
  - TELNET and TFTP
