### 부동산 감정평가 3방식

1. 원가방식

- 원가법 : 토지가치 + 건물잔여가치 평가
  ex) 토지 1억 + 건물가 1억 5천 : 2억 5천의 가치. 꼬마 빌딩 등에 주로 적용됨.

2. 비교방식

- 거래사례비교법 : 인근 유사 거래사례 비교평가 방식
  ex) 같은 동 아파트가 13억에 팔리면, 우리 아파트도 13억의 가치라고 판단함. 아파트 등에 주로 적용됨.

3. 수익방식

- 수익환원법 : 임대료와 수익률로 환산평가함.
  ex) 상가, 오피스텔 등
