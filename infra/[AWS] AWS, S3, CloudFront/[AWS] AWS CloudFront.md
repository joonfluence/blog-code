# 본론



### 관련 키워드

- Content Delivery Network(CDN) : 실제론 데이터센터. 
- Origin Server : 일반적으로 원본 데이터를 보관하고 있는 스토리지, S3 등을 말함.
- Edge Server = Edge Location : AWS에서 실질적으로 제공하는 전 세계에 퍼져있는 서버를 말함.
