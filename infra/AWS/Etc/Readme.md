### 탄력적 IP 주소

- EC2 인스턴스를 STOP하고 상태가 중지 상태에서 새롭게 실행 상태로 변경하면, 기존에 할당받은 IP주소가 변경되어 버리는 현상이 생겨버린다.
- 탄력적 IP란 한 번 할당 받으면 절대 바뀌지 않은 IP 주소를 말함.
- 실행 중인 인스턴스에 연결된 IP 주소 (EIP) 한 개는 무료로 사용.
